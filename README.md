# educatewebsite E-Learning un site web pour etudiants et enseignants pour avoir des cours en ligne. L'application fournit une plate-forme de cours en ligne facile à utiliser pour les étudiants et les enseignants. le professeur peut ajouter les cours ,
l'etudiant peut s'authentifier grace à une page login de l'étudiant en cliquant sur s'inscrire et automatiquement va y acceder à une page nommé profil pour avoir ses informations :
Les étudiants peuvent accéder aux cours en ligne à partir de n'importe quel endroit et à tout moment, ce qui leur permet de suivre les cours à leur propre rythme et selon leur propre emploi du temps. Les cours en ligne sont présentés sous forme de vidéos .
Les enseignants peuvent également utiliser l'application pour créer et partager des cours en ligne avec leurs étudiants.
